### Sets
Sets are used to store multiple items within a single variable. Sets a unordered, unchangeable, no duplicates are allowed, and unindexed.

``` Python
>>> my_set = {"salut", "hello", "hola", "ciao", "hola"}
>>> print(my_set)
{'ciao', 'salut', 'hello', 'hola'}

```
